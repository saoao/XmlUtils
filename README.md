# XmlUtils
Based on qt xml module,CRUD(Create, Read, Update and Delete) xml file.

基于qt xml模块，对xml文件进行增删改查。

编译环境：

| 库或工具                       | 描述     |
| ------------------------------ | -------- |
| Windows7（或以上）64操作系统   | 操作系统 |
| Qt5（包含msvc2017_64库文件）   | qt框架   |
| qt-vsaddin-msvc2017-2.3.2.vsix | vs插件   |
| Visual Studio 2017             | IDE      |
